An action set represents a set of actions, usually for a single player. This class must be subclassed to be used. An action set can contain both explicit, bindable single value actions (for example, \char`\"{}\+Jump\char`\"{}, \char`\"{}\+Left\char`\"{} and \char`\"{}\+Right\char`\"{}) and implicit, aggregate actions which combine together other actions into one or two axes, for example \char`\"{}\+Move\char`\"{}, which might consist of \char`\"{}\+Left\char`\"{}, \char`\"{}\+Right\char`\"{}, \char`\"{}\+Up\char`\"{} and \char`\"{}\+Down\char`\"{} filtered into a single two-\/axis control with its own applied circular deadzone, queryable vector value, etc.

\doxysubsection{Detailed Description}
An action set represents a set of actions, usually for a single player. This class must be subclassed to be used. An action set can contain both explicit, bindable single value actions (for example, \char`\"{}\+Jump\char`\"{}, \char`\"{}\+Left\char`\"{} and \char`\"{}\+Right\char`\"{}) and implicit, aggregate actions which combine together other actions into one or two axes, for example \char`\"{}\+Move\char`\"{}, which might consist of \char`\"{}\+Left\char`\"{}, \char`\"{}\+Right\char`\"{}, \char`\"{}\+Up\char`\"{} and \char`\"{}\+Down\char`\"{} filtered into a single two-\/axis control with its own applied circular deadzone, queryable vector value, etc.
